crystalline
===========

`crystalline` is a collection of containers & algorithms for the [crystal](https://github.com/manastech/crystal) programming language.

- Containers
  - Deque
  - Stack
  - Queue
  - Red-Black Trees
  - Splay Trees
  - Tries
  - Suffix Array
  - _k_-d Trees
  - Heaps __(NYI)__
  - Priority Queue __(NYI)__
- Algorithms __(NYI)__
